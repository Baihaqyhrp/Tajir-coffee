Langkah 1 — Buat Google Sheet

Buka sheets.google.com → buat spreadsheet baru → beri nama "Laporan Tajir Coffee".

Langkah 2 — Buka Apps Script

Di Google Sheets klik menu Ekstensi → Apps Script. Hapus kode yang ada, paste kode berikut:

function doPost(e) { var sheet = SpreadsheetApp .getActiveSpreadsheet() .getSheetByName("Laporan"); if (!sheet) { sheet = SpreadsheetApp .getActiveSpreadsheet() .insertSheet("Laporan"); sheet.appendRow([ "Tanggal","Menu","Laku","Stok", "Harga Satuan","Subtotal", "Total Kotor","Biaya","Profit" ]); } var data = JSON.parse(e.postData.contents); data.rows.forEach(function(row) { sheet.appendRow(row); }); return ContentService .createTextOutput(JSON.stringify({status:"ok"})) .setMimeType(ContentService.MimeType.JSON); }

Langkah 3 — Deploy sebagai Web App

1. Klik tombol Deploy → New Deployment

2. Pilih tipe: Web App

3. "Execute as": pilih Me

4. "Who has access": pilih Anyone

5. Klik Deploy → salin URL yang muncul
